# Iteración 4: Evaluación y Calificación

## Codificación

### Implementación del Sistema de Evaluación y Calificación
Se desarrollaron los endpoints de la API y los componentes de la interfaz de usuario para permitir a los docentes crear, gestionar y asignar evaluaciones y calificaciones.

**Archivos de la API Relevantes:**
A continuación se detallan los archivos de la API correspondientes a esta iteración:

*   **`src/app/api/evaluations/route.ts`**: Gestiona la creación de evaluaciones.
*   **`src/app/api/evaluations/[evaluationId]/route.ts`**: Gestiona la obtención, actualización y eliminación de una evaluación específica.
*   **`src/app/api/projects/[projectId]/evaluations/route.ts`**: Gestiona las evaluaciones dentro de un proyecto.

### Entregables
- **Código Fuente Comentado:** El código se encuentra en el directorio `src/app/api/evaluations`.
- **Módulo de Evaluación Funcional:** El sistema de evaluación y calificación está operativo.
