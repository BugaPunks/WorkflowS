## ADDED Requirements

### Requirement: Admin User Dashboard
Admins SHALL have a dedicated dashboard for platform management.

#### Scenario: Admin Overview
- **WHEN** admin logs in
- **THEN** they see platform statistics and quick access to management tools
- **AND** navigation to user management and system settings

### Requirement: Platform Statistics
The admin dashboard SHALL display key metrics about the platform usage.

#### Scenario: View Statistics
- **WHEN** admin accesses dashboard
- **THEN** they see counts of users, projects, active sprints
- **AND** recent activity and system health indicators</content>
