# Pruebas de la Iteración 3: Seguimiento y Kanban

## Pruebas Unitarias

- **Creación de Tareas:** Se crearán pruebas unitarias para verificar que la creación de tareas a través de la API funciona correctamente.
- **Actualización de Estado de Tareas:** Se probará que la actualización del estado de las tareas funciona como se espera.

## Pruebas de Integración

- **Ciclo de Vida de la Tarea:** Se realizarán pruebas de integración para asegurar que se puede crear una tarea, asignarla a un usuario, y luego actualizar su estado o eliminarla.
- **Sincronización del Tablero Kanban:** Se probará que el tablero Kanban se actualiza correctamente cuando se realizan cambios en las tareas.

## Pruebas de Interfaz de Usuario

- **Tablero Kanban:** Se probará la interfaz del tablero Kanban para verificar que las tareas se muestran correctamente y que la funcionalidad de arrastrar y soltar funciona como se espera.
- **Actualización de Tareas:** Se probará que los usuarios pueden actualizar los detalles de una tarea desde el tablero Kanban.
