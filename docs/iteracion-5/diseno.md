# Iteración 5: Dashboard y Métricas

## Diseño

### Diseño del Dashboard para Docentes
El dashboard se diseñó como una página principal para los docentes, con widgets que muestran:
- Un resumen de los proyectos activos.
- Gráficos de Burndown de los Sprints en curso.
- Una tabla con la contribución individual de los estudiantes.
- Alertas y notificaciones importantes.

*Nota: Los prototipos se encuentran en Figma.*

### Modelado de Visualizaciones de Métricas
Se diseñaron varios tipos de gráficos para visualizar las métricas:
- **Gráfico de Burndown:** Para el progreso del Sprint.
- **Gráfico de Barras:** Para la contribución individual de los estudiantes (tareas completadas).
- **Gráfico Circular:** Para la distribución de tareas por estado.

### Diseño de Interfaz de Reportes
Se diseñó una interfaz simple que permite a los docentes seleccionar un proyecto y generar un reporte en PDF con las métricas clave.

### Entregables
- **Prototipos de Interfaz:** Enlace a los diseños en Figma.
- **Esquemas de Visualización:** Documento que describe cada una de las visualizaciones de métricas.
