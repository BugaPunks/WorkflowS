# Pruebas de la Iteración 5: Dashboard y Métricas

## Pruebas Unitarias

- **Cálculo de Métricas:** Se crearán pruebas unitarias para verificar que el cálculo de las métricas del dashboard (burndown chart, velocidad, etc.) es correcto.
- **Generación de Reportes:** Se probará que la generación de reportes en diferentes formatos (CSV, JSON, etc.) funciona como se espera.

## Pruebas de Integración

- **Actualización del Dashboard:** Se realizarán pruebas de integración para asegurar que el dashboard se actualiza en tiempo real cuando se producen cambios en los datos.
- **Exportación de Datos:** Se probará que la exportación de datos desde el dashboard funciona correctamente y que los archivos generados son válidos.

## Pruebas de Interfaz de Usuario

- **Visualización de Gráficos:** Se probará que los gráficos y otras visualizaciones de datos en el dashboard se muestran correctamente y son fáciles de entender.
- **Panel de Exportación:** Se probará el panel de exportación para verificar que los usuarios pueden seleccionar el formato y el tipo de datos a exportar de forma intuitiva.
