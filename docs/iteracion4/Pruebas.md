# Pruebas de la Iteración 4: Evaluación y Calificación

## Pruebas Unitarias

- **Creación de Evaluaciones:** Se crearán pruebas unitarias para verificar que la creación de evaluaciones a través de la API funciona correctamente.
- **Cálculo de Calificaciones:** Se probará que el cálculo de calificaciones se realiza de acuerdo a las reglas definidas.

## Pruebas de Integración

- **Ciclo de Vida de la Evaluación:** Se realizarán pruebas de integración para asegurar que se puede crear una evaluación, asignarla a un estudiante, y luego verla y exportarla.
- **Notificaciones de Evaluación:** Se probará que se envían notificaciones a los estudiantes cuando se registra una nueva evaluación.

## Pruebas de Interfaz de Usuario

- **Formulario de Evaluación:** Se probará el formulario de evaluación para verificar que los docentes pueden registrar las calificaciones y la retroalimentación correctamente.
- **Vista de Calificaciones:** Se probará que los estudiantes pueden ver sus calificaciones y la retroalimentación de forma clara y precisa.
- **Exportación de Calificaciones:** Se probará que la funcionalidad de exportación de calificaciones genera un archivo con el formato y los datos correctos.
