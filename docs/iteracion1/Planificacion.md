# Planificación de la Iteración 1: Gestión de Equipos y Roles

## Historias de Usuario

- Como administrador, quiero poder crear nuevos usuarios en la plataforma.
- Como docente, quiero poder crear proyectos y asignarles estudiantes.
- Como estudiante, quiero poder ver los proyectos en los que estoy inscrito.

## Tareas Específicas

- Diseñar el modelo de datos para usuarios, proyectos y roles.
- Implementar la API para la creación de usuarios.
- Implementar la API para la creación de proyectos.
- Implementar la API para la asignación de roles a los usuarios en los proyectos.
- Crear la interfaz de usuario para la gestión de usuarios y proyectos.
