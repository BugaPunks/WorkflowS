# Planificación de la Iteración 3: Seguimiento y Kanban

## Historias de Usuario

- Como miembro del equipo, quiero poder ver todas mis tareas en un tablero Kanban.
- Como miembro del equipo, quiero poder actualizar el estado de mis tareas arrastrándolas y soltándolas en el tablero Kanban.
- Como scrum master, quiero poder ver el progreso del sprint en el tablero Kanban.

## Tareas Específicas

- Diseñar el modelo de datos para las tareas.
- Implementar la API para la creación, actualización y eliminación de tareas.
- Crear la interfaz de usuario para el tablero Kanban.
- Implementar la funcionalidad de arrastrar y soltar para actualizar el estado de las tareas.
- Integrar el tablero Kanban con los datos del sprint.
