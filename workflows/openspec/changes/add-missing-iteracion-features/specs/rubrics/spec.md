## ADDED Requirements
### Requirement: Advanced Rubric System
The system SHALL provide an advanced rubric builder as specified in "Ideas Adicionales".

#### Scenario: Create custom rubric
- **WHEN** a docente accesses the rubric builder
- **THEN** they can create multiple criteria with performance levels
- **AND** each level can have point values and descriptive feedback
- **AND** they can save rubrics as templates for reuse
- **AND** rubrics can be categorized (Documentation, Presentation, Code Review)

#### Scenario: Apply rubric to evaluation
- **WHEN** a docente evaluates a sprint or project deliverable
- **THEN** they can select a rubric template or create custom criteria
- **AND** they score each criterion using defined performance levels
- **AND** system calculates total score automatically
- **AND** they can add specific feedback for each criterion

#### Scenario: View rubric results
- **WHEN** a student views their evaluation results
- **THEN** they see detailed rubric breakdown with scores
- **AND** they can read feedback for each criterion and level
- **AND** they can compare their performance against rubric standards
- **AND** they can view historical rubric evaluations to track progress
