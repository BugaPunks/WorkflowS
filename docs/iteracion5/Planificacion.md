# Planificación de la Iteración 5: Dashboard y Métricas

## Historias de Usuario

- Como docente, quiero tener un dashboard con una vista general del progreso de todos los proyectos.
- Como scrum master, quiero tener un dashboard con métricas clave del sprint, como el burndown chart y la velocidad del equipo.
- Como miembro del equipo, quiero tener un dashboard con mis tareas asignadas y mi progreso.

## Tareas Específicas

- Diseñar el modelo de datos para el historial de exportaciones.
- Implementar la API para obtener los datos necesarios para los dashboards.
- Crear los componentes de la interfaz de usuario para los dashboards de cada rol.
- Integrar las bibliotecas de gráficos para visualizar las métricas.
- Implementar la funcionalidad de exportación de datos del dashboard.
