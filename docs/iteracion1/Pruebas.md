# Pruebas de la Iteración 1: Gestión de Equipos y Roles

## Pruebas Unitarias

- **Validación de Roles:** Se crearán pruebas unitarias para verificar que la asignación de roles a los usuarios funciona correctamente.
- **Creación de Usuarios:** Se probará que la creación de usuarios a través de la API funciona como se espera, incluyendo casos de éxito y de error.

## Pruebas de Integración

- **Creación de Proyectos y Asignación de Usuarios:** Se realizarán pruebas de integración para asegurar que se pueden crear proyectos y asignar usuarios a ellos con los roles correctos.

## Pruebas de Interfaz de Usuario

- **Formulario de Creación de Usuarios:** Se probará el formulario de creación de usuarios para asegurar que la validación de los campos es correcta y que los usuarios se crean correctamente.
- **Gestión de Proyectos:** Se probará la interfaz de gestión de proyectos para verificar que los docentes pueden crear proyectos y asignar estudiantes a ellos.
