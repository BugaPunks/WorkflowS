## MODIFIED Requirements

### Requirement: Clean Directory Structure
The project SHALL maintain a single, clean source directory structure.

#### Scenario: No Duplicate Directories
- **WHEN** developers navigate the codebase
- **THEN** they find files in expected locations
- **AND** no duplicate or confusing directory structures exist

#### Scenario: Consistent File Organization
- **WHEN** adding new files to the project
- **THEN** they follow the established directory patterns
- **AND** are placed in src/ with appropriate subdirectories</content>
