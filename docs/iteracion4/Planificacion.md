# Planificación de la Iteración 4: Evaluación y Calificación

## Historias de Usuario

- Como docente, quiero poder evaluar a los estudiantes en cada sprint y al final del proyecto.
- Como estudiante, quiero poder ver mis calificaciones y la retroalimentación de mis evaluaciones.
- Como docente, quiero poder exportar las calificaciones de un proyecto.

## Tareas Específicas

- Diseñar el modelo de datos para las evaluaciones.
- Implementar la API para la creación y gestión de evaluaciones.
- Crear la interfaz de usuario para que los docentes puedan registrar las evaluaciones.
- Crear la interfaz de usuario para que los estudiantes puedan ver sus calificaciones.
- Implementar la funcionalidad de exportación de calificaciones.
